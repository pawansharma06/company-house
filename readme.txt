== Description ==
Company House is a plugin design and builds to enable company search directly on your website with a simple shortcode using Gov.UK Official API.

Company House will be fully supported and maintained until at least 2022, or as long as is necessary and API provided by Gov.UK

== 3rd Party Service ==
1. Company House API Key is required by the plugin to work.
2. Company House Plugin totally depends upon https://api.companieshouse.gov.uk/ Api to Work and fetch result.
3. Plugin Doesn\'t store any client information. 
4. Information provided is license under [https://www.nationalarchives.gov.uk/doc/open-government-licence/version/3/]




== Installation ==
1. Upload the entire `company-house` folder to the `/wp-content/plugins/` directory.
1. Activate the plugin through the \'Plugins\' menu in WordPress.

You will find \'company house\' menu in your WordPress admin panel.

For basic usage, you can also have a look at the [plugin web site](https://planetarymarketing.in/).
